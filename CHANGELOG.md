# 牛津高阶英汉双解词典 Web 版 - 变更日志

> ## 📋 项目管理规范（强制）
>
> 本项目采用 **AI 辅助开发模式**，遵循以下强制规范：
>
> 1. **每次修改必须记录** — 任何代码变更、配置调整、文档更新，均须在 `CHANGELOG.md` 中追加条目
> 2. **条目格式** — 遵循语义化版本 `[版本号] - 日期`，分类使用 emoji 标记（🚀 新功能 / 🔧 优化 / 🐛 修复 / 📝 文档）
> 3. **变更描述** — 清晰说明改动内容、涉及文件和原因
> 4. **Git 提交关联** — 每次 `CHANGELOG.md` 更新应与 `git commit` 一同推送
> 5. **违反后果** — 未记录变更视为不合规，需立即补录

> 以上规则自 `v1.0.0` 起强制执行。

## [1.0.0] - 2026-06-17

### 🚀 项目初始化

#### 架构设计
- **数据层**: MDX 词典文件 → SQLite 数据库（283,816 词条）
- **服务层**: Python 内置 http.server，零外部依赖
- **前端层**: 纯 HTML + CSS + JS，保留原词典 oald10.css/oald10.js 样式

#### 已实现功能

##### 后端
- [x] Python MDX 解析脚本 (`scripts/build_db.py`)
- [x] HTTP 服务 (`server.py`) - 端口 18310
- [x] 搜索 API: `GET /api/search?q=word` — 前缀匹配 + 精确优先
- [x] 查词 API: `GET /api/lookup?word=hello` — 返回完整 HTML
- [x] 自动补全 API: `GET /api/suggest?q=hel` — 实时建议
- [x] `entry://` 跨词链接转换
- [x] `sound://` 发音链接处理

##### 前端
- [x] 极简搜索界面，粘性顶部搜索栏
- [x] 实时自动补全下拉列表
- [x] 保留原词典 oald10.css 排版样式
- [x] 保留原词典 oald10.js 交互效果
- [x] 响应式布局，支持移动端
- [x] URL 参数支持 (`?q=hello`)
- [x] 快捷示例词按钮
- [x] 跨词条点击跳转

##### 部署
- [x] Mac 开机自启动 (launchd plist)
- [x] 一键安装/卸载脚本 (`start.sh install`)

## [1.1.0] - 2026-06-17

### 🚀 新功能

- **局域网访问**: 服务器绑定从 `127.0.0.1` 改为 `0.0.0.0`，允许同 WiFi 下设备访问
- **iPhone 支持**: 启动时自动显示本机局域网 IP，iPhone 浏览器直接访问
- 显示示例: `http://192.168.x.x:18310`
- **Safari Tab 图标**: 新增 `data/favicon.svg`，蓝色书本文案搭配「OX」标识
